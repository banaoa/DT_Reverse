Commons Logging is a thin adapter allowing configurable bridging to other, well known logging systems. Version 99.0-does-not-exist is a dummy implementation with actually does nothing, and most of all: it has no dependencies.

COMMONS-LOGGING VERSION 99.0-does-not-exist IS NOT IN ANY WAY AFFILIATED WITH THE ORIGINAL DEVELOPERS OF COMMONS-LOGGING NOR WITH APACHE.

See http://www.day-to-day-stuff.blogspot.com/2007/07/no-more-commons-logging for information on how to use no-commons-logging.
